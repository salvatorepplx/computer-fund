# Strategy status ladder — nothing is relied upon until earned

The fund does NOT trust a thesis because it was generated. Generation is cheap and worthless
on its own (random tuples = curve-fit-by-volume). A coordinate must CLIMB this ladder on evidence:

1. **candidate_unvetted** — a coordinate in the search space (auto-sampled tuple OR a raw idea).
   Carries ZERO weight. Most die here. Generating these fast is NOT progress.

2. **researched** — Computer/Teammate did real investigation: is there a credible mechanism? prior
   literature? actual data availability? a plausible economic reason it could work? A coordinate
   with no mechanism is killed here, not promoted. (Requires a written rationale + sources.)

3. **testing** — passed the smell test; now accumulating real data through the falsification harness
   (min_n, permutation null p<=0.10, circularity guard, cross-sectional generalization >=30% of universe).

4. **edge** — survived ALL falsifiers on out-of-sample/forward data. ONLY an `edge` thesis can produce
   a PROPOSED trade. This is the only status the alpha pipeline will act on.

5. **killed** — failed any gate. Recorded in runs/CORPSES.md with the reason. A kill is a win.

Rules:
- The number of candidate_unvetted entries is NOT a measure of breadth or progress.
- Real breadth = number of theses at `researched`+ with DISTINCT, independent mechanisms.
- Never describe unvetted coordinates as "the strategies we're running." They are hypotheses to kill.
- A thesis advances one rung only with explicit evidence logged. No silent promotions.
